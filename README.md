# Super Store Sales Dashboard | Power BI

## Project Overview

This project is an interactive Sales Analytics Dashboard developed in Power BI using Super Store sales data. The dashboard provides insights into sales performance, profitability, customer segments, shipping modes, payment methods, regional sales distribution, and product categories.

The goal of this project is to transform raw sales data into meaningful business insights through data modeling, visualization, and dashboard design.

---

## Objectives

- Analyze overall sales performance
- Track profit and order trends
- Compare sales across regions
- Identify top-performing product categories
- Understand customer segment contribution
- Monitor sales growth over time

---

## Tools & Technologies

- Power BI Desktop
- Power Query
- DAX
- Data Modeling
- Data Visualization

---

## Dashboard Highlights

### Key KPIs
- Total Sales: 1.6M
- Total Profit: 175K
- Total Quantity Sold: 22K
- Total Orders: 6K

### Analysis Included
- Sales by Region
- Sales by Category
- Sales by Sub-Category
- Sales by Ship Mode
- Sales by Segment
- Sales by Payment Mode
- Monthly Sales Trends
- Year-over-Year Sales Comparison
- Profit Analysis by State

---

## Key Insights

- The West region generated the highest sales contribution.
- Consumer segment accounted for the largest share of sales.
- COD was the most preferred payment method.
- Phones, Chairs, and Binders were among the top-performing sub-categories.
- Standard Class was the most frequently used shipping mode.
